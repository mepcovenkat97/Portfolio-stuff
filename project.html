Money Tracker

This application tracks the expenses made by us. Initial budget should be updated and the expenses made are tracked using that. The expenses can be classified into categories. We can also add categories. Category-wise Expenses made are represented in pie-charts. Also we can update the expense or delete it. Pagination also implemented while displaying the expenses. Authentication has been implemented using Web Tokens.
